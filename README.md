# Orphan Destroyer
Small disclaimer: The name was not my idea.

Simple discord bot created for a personal server using node.js

Setup:

1: `npm install`

2: Create `auth.json` with the following contents:

```json
{
	"token": "<your bot token>",
	"clientId": "<bot client id>",
	"guildId": "<server id>"
}
```

3: `node index.js`

4: Profit
